Step 1: Installation - The Right Way üîß

No "easy" installers. We do this manually. We understand every step.

1.1 Boot and Network

# Boot from Arch ISO
# Connect to WiFi:
iwctl
station wlan0 scan
station wlan0 connect [SSID]
ping -c 3 archlinux.org

1.2 Partitioning for Performance

# UEFI systems only (you should be using UEFI)
lsblk
gdisk /dev/nvme0n1

# Partitions:
# 1: 512M EF00  (/boot/efi)
# 2: 2G   8200  (swap - 2x RAM for hibernate)
# 3: 30G  8304  (/)
# 4: rest 8302  (/home)

1.3 Format and Mount

mkfs.fat -F32 /dev/nvme0n1p1
mkswap /dev/nvme0n1p2
mkfs.ext4 /dev/nvme0n1p3
mkfs.ext4 /dev/nvme0n1p4

mount /dev/nvme0n1p3 /mnt
mkdir -p /mnt/boot /mnt/home
mount /dev/nvme0n1p1 /mnt/boot
mount /dev/nvme0n1p4 /mnt/home
swapon /dev/nvme0n1p2

1.4 Install Base System

pacstrap /mnt base base-devel linux linux-firmware vim git
genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt

Step 2: System Configuration ‚öôÔ∏è

2.1 Timezone and Locale

ln -sf /usr/share/zoneinfo/Region/City /etc/localtime
hwclock --systohc
vim /etc/locale.gen  # Uncomment en_US.UTF-8
locale-gen
echo "LANG=en_US.UTF-8" > /etc/locale.conf

2.2 Hostname and Network

echo "arch-beast" > /etc/hostname
# Add to /etc/hosts:
127.0.0.1   localhost
::1         localhost
127.0.1.1   arch-beast.localdomain arch-beast

pacman -S networkmanager
systemctl enable NetworkManager

2.3 Users and Bootloader

passwd  # Set root password
useradd -m -G wheel -s /bin/bash yourusername
passwd yourusername

# Enable sudo
EDITOR=vim visudo  # Uncomment %wheel ALL=(ALL) ALL

# systemd-boot
bootctl install

Step 3: Performance Optimization ‚ö°

3.1 Microcode

# Intel:
pacman -S intel-ucode

# AMD:
pacman -S amd-ucode

3.2 Enable TRIM for SSDs

systemctl enable fstrim.timer

3.3 Tweak Swappiness

echo "vm.swappiness=10" > /etc/sysctl.d/99-swappiness.conf

3.4 CPU Governor (Performance Mode)

pacman -S cpupower
systemctl enable cpupower
cpupower frequency-set -g performance

Step 4: The AUR - Your Weapon üó°Ô∏è

git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
cd ..
rm -rf yay

# Now install anything:
yay -S visual-studio-code-bin spotify slack-desktop

Step 5: Development Environment üíª

5.1 Essential Tools

5.2 Docker Setup

systemctl enable docker

6.2 Clean Cache

config config --local status.showUntrackedFiles no
